
gi Library for html5
--------------------
R.Hoetzlein (c) 2023, Quanta Sciences

2D GUI library written in 
javascript to work with html5 canvas.
Features: network auto-build, reactive layout

gxLib is not here. gx is lower-level, where
the gx feature set is support by html5 canvas
draw commands. gxLib only written for C++/OpenGL.

giLib.js - GUI library with auto-layout

giVis.js - Quanta Thin Visual Client for html5/js
           running inside giGrid container

giGrid.js - primary region class for giLib

index.html - top-level test point and main display loop
